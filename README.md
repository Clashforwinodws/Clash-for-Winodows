## 🧭 如何选择 Clash 客户端？

⚠️ 停更客户端可能无法兼容新协议，建议迁移到活跃项目。

- **Windows**：推荐 FlClash 或 Clash Verge Rev
- **macOS**：推荐 FlClash 或 ClashX Meta
- **iOS**：推荐 Shadowrocket（小火箭）
- **Android**：推荐 FlClash 或 Clash Meta for Android
- **路由器/全屋**：OpenClash、PassWall2
