🚀 最新版本 v0.20.40

2026-06-30 发布

此版本为稳定性与兼容性更新，重点优化了 TUN 模式性能，修复了订阅更新偶发失败问题，并增强了对 Clash Meta 内核的兼容性。建议所有用户升级。

✨ 新增功能

  • 支持 Clash Meta 内核的 script 规则类型
  • 新增一键导出当前配置为 YAML 文件

⚡ 改进优化

  • TUN 模式下 CPU 占用降低约 15%
  • 订阅更新失败后自动重试，提升成功率
  • UI 内节点延迟测试速度更快

🐛 问题修复

  • 修复部分订阅链接因 User-Agent 被拒的问题
  • 解决 macOS Ventura 上菜单栏图标消失
  • 修复 Linux 下 systemd 服务重启后端口占用
